## Локальный запуск

### Требования
- [Docker Desktop](https://www.docker.com/products/docker-desktop/)
- Git

### Установка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/IBloodFoxI/delivery-service.git
cd delivery-service
```

2. Создайте файл `.env` на основе примера:
```bash
cp .env.example .env
```

3. Соберите и запустите контейнеры:
```bash
docker compose up -d --build
```

4. Дождитесь запуска всех контейнеров (1–2 минуты), затем откройте в браузере:
```
http://localhost/
```

При первом запуске автоматически создаются тестовые пользователи и товары.

### Остановка

```bash
docker compose down
```

## Стек технологий

- **Backend:** Python 3.11, Django 4.2.7
- **База данных:** PostgreSQL 15
- **Кэш / сессии:** Redis 7
- **Веб-сервер:** Nginx + Gunicorn
- **Контейнеризация:** Docker Compose
- **Оплата:** ЮКасса
- **Email:** Resend API
- **Карты:** Yandex Maps API
